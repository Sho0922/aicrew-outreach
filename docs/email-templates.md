# 業種別 営業メール テンプレート（aicrew）

`pnpm gen-emails` が `data/prospects.xlsx` の各社にこのテンプレートを差し込み、
`data/emails.csv`（優先度・法人名・業種・宛先・**件名**・**本文**）を生成します。

## 差し込み変数
- リスト由来: `法人名` `代表者` `業種` `自動化余地`（クローラが業種別に付与）
- 差出人（`.env.local` で設定）: `SENDER_NAME` `SENDER_COMPANY` `SENDER_ADDRESS` `SENDER_TEL` `SENDER_EMAIL` `OPTOUT_CONTACT`

```bash
# .env.local に追記（送信前に必須）
SENDER_NAME=五反田 翔
SENDER_COMPANY=aicrew（Baller株式会社）
SENDER_ADDRESS=東京都...
SENDER_TEL=03-xxxx-xxxx
SENDER_EMAIL=sho@aicrew.jp
OPTOUT_CONTACT=本メールへのご返信
```

## ⚠ 運用上の注意
- **特定電子メール法**: 送信者の氏名・住所の明示、オプトアウト（配信停止）導線が必須。本テンプレートのフッターに含むため、`SENDER_*` を必ず実値に。
- 年商・レガシー度は**推定**。送信前に各社の年商・業務実態を人手で検証。
- 一斉送信は1日あたりの件数・到達率に配慮し、段階的に。返信・配信停止の管理を徹底。
